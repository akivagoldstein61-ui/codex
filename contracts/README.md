# Contract Schemas

This directory holds canonical JSON Schemas for foundational Deck Factory OS contracts.

## Schemas
- `event-envelope.schema.json`
- `deck-spec.schema.json`
- `card-spec.schema.json`
- `journey-spec.schema.json`
- `content-validation-report.schema.json`
- `asset-manifest.schema.json`
- `entitlement-check-result.schema.json`

## Usage
Treat these schemas as the contract boundary for ingestion, validation, entitlement checks, and event traceability.
